# Portfolio Description

## Project Title

SAP Document & Compliance Master Data Quality Audit Tool

## Short Description

Python, SQL, and Streamlit tool that audits SAP-like document and compliance master data, detects data quality issues, calculates risk-based scores, and generates executive remediation reports.

## Professional Summary

This project simulates a SAP-like master data quality audit process focused on document and compliance records.

The tool uses synthetic SAP-like data to identify issues such as expired active documents, missing approvals, inactive owners, invalid date sequences, missing review cycles, missing file references, and incomplete ownership records.

The pipeline generates cleaned datasets, audit results, document-level data quality scores, risk categories, executive summaries, exception reports, and remediation backlogs.

## Business Value

The project demonstrates how organizations can reduce manual spreadsheet review, improve audit readiness, and prioritize corrective actions using Python, SQL, and dashboard automation.

It is especially relevant for compliance, quality, supply chain, operations, business systems, and document control teams.

## Skills Demonstrated

- SAP-like data analysis
- Data quality auditing
- Compliance data governance
- Python automation
- Pandas data processing
- SQL validation queries
- Risk scoring models
- Executive reporting
- Streamlit dashboard development
- Remediation backlog creation
- Documentation for portfolio presentation

## Interview Pitch

I built a SAP-like data quality audit tool focused on document and compliance master data. The project does not use vendor master, material master, real SAP data, or confidential company information.

It generates synthetic SAP-like records, applies documented audit rules, calculates document-level quality scores, classifies risk, and produces executive reports and a remediation backlog.

The goal was to simulate how compliance, quality, supply chain, or business systems teams could automate the review of document records instead of relying only on manual spreadsheet checks.
